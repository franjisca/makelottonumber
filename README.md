🎰 행운의 로또 번호 생성기
재미있고 예쁜 디자인의 로또 번호 생성기 웹사이트입니다.
✨ 주요 기능
1. 번호 생성

1-45 중 6개 번호 랜덤 생성
애니메이션 효과와 함께 생성
번호별 색상 구분 (1-10: 노랑, 11-20: 파랑, 21-30: 빨강, 31-40: 회색, 41-45: 초록)

2. 번호 저장 (쿠키 활용)

생성된 번호 세트 저장
브라우저 쿠키에 30일간 보관
재방문 시 자동으로 불러오기
중복 번호 저장 방지

3. 이미지로 저장

실제 로또 용지 스타일로 디자인
저장된 번호들을 PNG 이미지로 다운로드
고해상도(2x) 이미지 생성
발행일 자동 표시

4. 초기화 기능

저장된 모든 번호 세트 삭제
삭제 전 확인 모달 표시

5. 광고 영역

수익화를 위한 광고 배치
728x90 (상단 배너)
300x250 (하단 사각형 광고 2개)

🎨 디자인 특징

그라데이션 배경: 보라색, 분홍색, 인디고의 화려한 조합
애니메이션:

번호 생성 시 회전 애니메이션
번호 표시 시 바운스 효과
당첨 시 컨페티(색종이) 효과
배경 블롭 애니메이션


글래스모피즘: 반투명한 카드 디자인
반응형: 모바일부터 데스크톱까지 모든 화면 크기 지원

🛠️ 기술 스택

React 18 (TypeScript)
Tailwind CSS 3
Vite
Canvas API (이미지 생성)
